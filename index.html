BoJack Horseman API

Unofficial API for the animated Netflix series BoJack Horseman.

GitHub Repo

Endpoint

get https://bojack-api.herokuapp.com/api

Search By

all characters https://bojack-api.herokuapp.com/api/characters
search by id https://bojack-api.herokuapp.com/api/characters/Character Id
search by name https://bojack-api.herokuapp.com/api/characters/name/Character Name

Features

Property Description
id Id number of character [Number]
name Full name of character [string]
birthDate Birthdate of character [string]
birthPlace Birth place of character [string]
species Species of character [string]
voiceActor Voice actor of character [string]
image URL to image of character [string]
quote Quote from character [string]
